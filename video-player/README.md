# Video Player

Webブラウザベースの動画プレイヤー

## 必要条件

- Python 3.8+
- Flask
- Pillow
- OpenCV

## インストール

```bash
git clone https://github.com/sohei-t/python-video-tools.git
cd python-video-tools/video-player
pip install flask pillow opencv-python
```

## 使用方法

```bash
# カレントディレクトリの動画を再生
python video_player.py

# 指定ディレクトリの動画を再生
python video_player.py -d ./videos

# ポートを指定
python video_player.py --port 9000

# ブラウザを自動で開かない
python video_player.py --no-browser
```

## オプション

| オプション | 説明 | デフォルト |
|-----------|------|-----------|
| `-d`, `--directory` | 動画ファイルのディレクトリ | カレントディレクトリ |
| `-p`, `--port` | サーバーのポート番号 | 8080 |
| `--host` | サーバーのホスト | localhost |
| `--no-browser` | ブラウザを自動で開かない | - |

## 機能

### サムネイル自動生成

初回起動時に動画からサムネイルを自動生成します。サムネイルは `thumbnails/` フォルダに保存されます。

### 再生速度変更

0.25x〜16x の範囲で再生速度を変更できます。

### キーボードショートカット

| キー | 機能 |
|-----|------|
| `←` `→` | 前後の動画に移動 |
| `Space` | 再生/一時停止 |
| `,` `<` | 再生速度を遅く |
| `.` `>` | 再生速度を速く |

### 連続再生

動画の再生が終わると自動的に次の動画を再生します。

## 対応フォーマット

`.mp4`, `.avi`, `.mov`, `.mkv`, `.webm`, `.flv`, `.wmv`, `.m4v`

## 注意事項

- 大量の動画がある場合、初回起動時のサムネイル生成に時間がかかることがあります
- ブラウザによっては一部の動画フォーマットが再生できない場合があります
