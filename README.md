# Python Video Tools

動画編集用のPythonツール集

## ツール一覧

| ディレクトリ | 説明 |
|-------------|------|
| [video-segment-tools](./video-segment-tools/) | 動画の切り出し・分割・結合ツール |
| [video-grid-composer](./video-grid-composer/) | 複数動画の横並び・縦並び・グリッド結合ツール |

## 必要条件

- Python 3.8+
- ffmpeg（video-segment-tools用）
- OpenCV, NumPy（video-grid-composer用）
